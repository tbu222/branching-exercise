What git command creates a branch?
git branch <branch>

What is the difference between a fast-forward and recursive merge?
fast forward: No new commits on the master	
recursive: New commits on the master

What git command changes to another branch?
git checkout [<branch>]

What git command deletes a branch?
git branch --delete <branchname>

How do merge conflicts happen?
when people make different changes to the same line of the same file, or when one person edits a file and another person deletes the same file
